Marketlens on GitHub

Marketlens collects and serves historical order book data for Polymarket prediction markets: tick level L2 snapshots and deltas, trades, candles, and Parquet exports, with a backtesting engine that simulates queue priority, latency, and slippage.

  • marketlens-python: the open source Python SDK, backtest engine, and MCP server. Install with pip install marketlens.
  • marketlens on PyPI: the released package, supported on Python 3.10 through 3.13.
  • Documentation: getting started, data reference, and backtesting guides.
