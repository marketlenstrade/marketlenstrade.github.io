Marketlens · Open Source

Polymarket order book history, replayed tick by tick.

Marketlens collects the full L2 order book of Polymarket prediction markets and serves it as tick level snapshots and deltas, executed trades, candles, and Parquet exports. The open source Python SDK reconstructs books at any millisecond and backtests strategies against them with realistic execution: queue priority, latency, and slippage.

Quick Start

# pip install marketlens
from marketlens import MarketLens

client = MarketLens()
markets = client.markets.list(
    q="bitcoin", status="active",
    sort="-liquidity", take=5,
)
for m in markets:
    print(m.question, m.outcomes[0].last_price)

Free API keys at marketlens.trade; the Getting Started guide covers auth, order books, and a first backtest.

What's Inside

The data and the tools

  • Order books: L2 snapshots plus incremental deltas, reconstructable at millisecond resolution, nothing interpolated.
  • Trades and candles: executed trades and OHLCV aggregations across every collected market.
  • Backtesting: subclass Strategy hooks (on_book, on_trade, on_fill) and run against historical books with simulated queue position, latency, slippage, and fees.
  • Exports: bulk Parquet downloads that feed straight into offline backtest runs.
  • MCP server: hand the same data and backtest engine to AI agents over the Model Context Protocol.

Order book history starts on 2026-03-01, with continuous coverage since 2026-04-13.
